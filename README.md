# 📌 Student Performance Prediction using Machine Learning

## 📖 Overview

This project predicts student academic performance using machine learning techniques. It analyzes factors like study time, attendance, previous grades, and personal background to estimate final exam scores. The system helps identify students who may need academic support and enables early intervention.

---

## 🎯 Objectives

* Analyze the relationship between student attributes and performance
* Clean and preprocess data for accurate analysis
* Visualize patterns and correlations
* Build and compare machine learning models
* Evaluate model performance using standard metrics
* Provide insights to improve student outcomes

---

## ❗ Problem Statement

Educational institutions face difficulty in identifying weak students early. Manual evaluation of multiple factors is inefficient. This project solves the problem by using machine learning to automatically predict student performance.

---

## 🧠 Methodology

1. **Dataset Collection**

   * Student dataset with academic and demographic details

2. **Data Preprocessing**

   * Handling missing values
   * Encoding categorical data
   * Feature scaling

3. **Exploratory Data Analysis (EDA)**

   * Identifying patterns and correlations
   * Data visualization using graphs and heatmaps

4. **Feature Selection**

   * Selecting important features

5. **Model Development**

   * Linear Regression
   * Random Forest Regression
   * Decision Tree Regression

6. **Model Evaluation**

   * RMSE, MAE, R² Score

7. **Prediction**

   * Predicting final student grades

---

## 📊 Key Insights

* Previous grades strongly influence final performance
* Study time improves results
* Absences negatively affect performance
* Family support impacts academic outcomes

---

## 🏗️ System Architecture

* **Input Layer:** Dataset (CSV file)
